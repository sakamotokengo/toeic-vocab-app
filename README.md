# TOEIC単語トレーナー

TOEIC頻出英単語を4択クイズで学習できる、無料のブラウザアプリです。
サーバー不要の静的サイト（HTML / CSS / JavaScript）で、学習データはブラウザのlocalStorageに保存されます。

## 特長

- TOEIC頻出1100語（基礎〜600点 / 中級〜730点 / 上級860点〜）を収録。`wordlist.html` に全単語の一覧ページあり
- 1ラウンド10問の4択クイズで、正答率をスコア化
- 回答後は例文（英文＋和訳）と覚え方の豆知識を表示。他の選択肢をクリックすると、その単語の解説も見られる
- 間違えた単語は次のラウンドに自動で再出題。正解後も3・6・12ラウンド後に復習出題（間隔反復）
- 学習記録から推定語彙力レベル・レベル別習得状況・スコア推移グラフを表示

## 使い方

`index.html` をブラウザで開くだけで動きます。

## 単語データの追加

単語はすべて `words.js` にあります。最後の `];` の手前に次の形式で1行追加してください。

```js
{w:"英単語", m:"日本語の意味", ex:"英語の例文", exJa:"例文の和訳", tip:"覚え方や用途の豆知識", lv:2},
```

`lv` は 1=基礎（〜600点）/ 2=中級（〜730点）/ 3=上級（860点〜）。
生成AIでデータを量産する場合のプロンプトは `word-prompt.md` を参照してください。
